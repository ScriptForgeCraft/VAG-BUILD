# VAG Armenia Renovation Landing

Marketing landing page for renovation services in Armenia with:

- multilingual UI (`am`, `ru`, `en`)
- interactive renovation calculator
- WhatsApp lead capture
- static Vite build

## Commands

```bash
npm install
npm run dev
npm run build
npm test
```

## Project Notes

- Active calculator pricing is stored in `src/features/calculator/model/pricing.js`.
- General calculator texts and field definitions are stored in `src/features/calculator/model/constants.js`.
- Placeholder visuals are generated locally, so the UI does not depend on external demo image services.
